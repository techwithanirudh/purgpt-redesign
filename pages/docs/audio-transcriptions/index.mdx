# What are Audio Transcriptions?

<Callout type="warning" emoji="⚠️">
    Audio transcriptions are coming soon. The following docs are not currently reliable!
</Callout>

Audio transcriptions (or simply transcriptions) are used to flip audio into text. This technology is particularily useful for:

- Voice-activated commands
- Dictation software
- Translation
- Assistants
- Subtitles

PurGPT offers OpenAI's awesome open-source [Whisper model](https://openai.com/blog/whisper/) for this. File uploads are currently limited to 25 MB and the following input file types are supported: `mp3`, `mp4`, `mpeg`, `mpga`, `m4a`, `wav`, and `webm`.
