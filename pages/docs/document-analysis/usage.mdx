# Using Document Analysis


## Request

**URL: `https://purgpt.xyz/hugging-face/documents/scan`**

Request method: `POST`

### TS Interface

```ts
interface Request {
	document: string;
	prompt: string;
};
```

### Example JSON Request Body

```json
{
	"document": "https://online.visual-paradigm.com/repository/images/4367fbb4-b023-4e4f-b910-8db451358a3b/recipe-cards-design/simple-pancake-recipe-card.png",
	"prompt": "What is the first step?"
}
```

## Response

### TS Interface

```ts
interface Response {
	id: string;
	object: 'document.ask';
	created: number;
	model: 'layoutlm-document-qa';
	choices: [
		{
			index: 0;
			result: {
				answer: string;
				score: number;
			};
			finish_reason: 'stop';
		};
	]
};
```
