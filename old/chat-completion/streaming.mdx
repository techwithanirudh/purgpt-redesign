# Streaming Completion Responses

<Callout type="warning" emoji="⚠️">
    Streaming is coming soon. The following docs are not currently reliable!
</Callout>

PurGPT supports the ability to stream responses. This not only improves response time, but also gives the typing feel like on ChatGPT.

To enable streaming, simply include `"stream": true{:js}` in your request body.

## Example Request Body With `stream`

```json {9}
{
    "model": "gpt-3.5-turbo",
    "messages": [
        {
            "role": "user",
            "content": "who are you? answer with 3 words"
        }
    ],
    "stream": true
}
```
