# Embeddings Models

PurGPT provides many of OpenAI's embedding models. Please refer to their [docs](https://platform.openai.com/docs/guides/embeddings) to understand each model specifically. Below is a table, with the corresponding description, for each model:

|          Title          |                                        Description                                       |
|-------------------------|-----------------------------------------------------------------------------------------|
|   gpt-3.5-turbo-wiki   | Provides embeddings based on the GPT-3.5 Turbo model trained on a mixture of licensed data, data created by human trainers, and publicly available data. |
| gpt-3.5-turbo-wiki-instruct   | Combines the capabilities of GPT-3.5 Turbo with instructions to perform specific tasks and generate informative responses. |
|       curie-wiki        | Offers embeddings based on the Curie model trained on a mixture of licensed data, data created by human trainers, and publicly available data. |
|     curie-wiki-instruct   | Combines the Curie model's embeddings with instructions to perform specific tasks and generate detailed responses. |
| text-embedding-ada-002 | Provides text embeddings based on the Ada model trained on a mixture of licensed data, data created by human trainers, and publicly available data. |
|  text-embedding-babbage-003 | Provides text embeddings based on the Babbage model trained on a mixture of licensed data, data created by human trainers, and publicly available data. |
| text-embedding-curie-003 | Offers text embeddings based on the Curie model trained on a mixture of licensed data, data created by human trainers, and publicly available data. |
|  text-embedding-davinci-003 | Provides text embeddings based on the Davinci model trained on a mixture of licensed data, data created by human trainers, and publicly available data. |
| text-embedding-davinci-002 | Provides text embeddings based on the Davinci model trained on a mixture of licensed data, data created by human trainers, and publicly available data. |
| text-embedding-davinci-001 | Provides text embeddings based on the Davinci model trained on a mixture of licensed data, data created by human trainers, and publicly available data. |
| text-embedding-codex | Offers embeddings based on the OpenAI Codex model trained on a mixture of licensed data, data created by human trainers, and publicly available data. |
| text-embedding-ada-003 | Provides text embeddings based on the Ada model trained on a mixture of licensed data, data created by human trainers, and publicly available data. |
| text-embedding-gpt3.5-turbo | Offers embeddings based on the GPT-3.5 Turbo model trained on a mixture of licensed data, data created by human trainers, and publicly available data. |
